# Typescript katas

## Developing

### Set up

```shell
nvm install # install the version of node configured in .nvmrc 
nvm use # use the version of node configured in .nvmrc 

npm i # Install node dependencies
```

### Run tests

```shell
npm t # Run tests once
npm run test:watch # Run in watch mode
```

### Linting

```shell
npm run lint
npm run lint:fix
```
